## Project Description

This project is a responsive personal portfolio website developed using HTML, CSS, and JavaScript. 

The website includes structured sections for About Me, Projects, and Contact, along with interactive features such as Dark Mode toggle and a dynamic greeting message based on the time of day.

The goal of this assignment is to practice front-end fundamentals, responsive design, and responsible AI-assisted development.

---

## Features

- Responsive layout (desktop, tablet, mobile)
- Modern purple-themed UI design
- Dark Mode toggle
- Time-based greeting message
- Interactive contact form (client-side)
- Clean and organized project structure

---

## Technologies Used

- HTML5
- CSS3 (Flexbox, Media Queries, CSS Variables)
- JavaScript (DOM manipulation, Event Listeners)
